\begin{frame}[c]{Is Bayesian Optimization the Ideal Method for HPO?}

Standard Bayesian optimization is:
\pause

\begin{itemize}
    \item built for sequential decision-making under uncertainty in which evaluations are expensive and the objective function is unknown,
    \pause
    \item theoretically grounded, well-established, and widely studied,
    \pause
    \item and based on standard smoothness assumptions, which makes it widely applicable.
    %\footnote{For example, some regret guarantees assume the objective lies in the RKHS induced by the chosen kernel.}
\end{itemize}

\vspace{0.5cm}
\pause
$\leadsto$ \textbf{Yes,} because it can be more sample-efficient than random search, more adaptive than grid search, and more systematic than manual search. \\
\pause
$\leadsto$ \textbf{But}, it might still be too inefficient for practical HPO tasks (e.g., where a single function evaluation requires training and evaluating a large neural network).

\end{frame}

\begin{frame}[c]{What Do We Know About HPO of ML Models?}

Some observations:
\pause

\begin{itemize}
    \item We often search the same space on different datasets. \\ $\leadsto$ Similar tasks might have similar objective functions.
    \pause
    \item Prior information is available from previous experiments or the domain. \\ $\leadsto$ We do not need to start from scratch.
    \pause
    \item Evaluating ML pipelines is iterative. \\ $\leadsto$ We can get low-cost approximations of the true, costly function.
\end{itemize}
\pause

\vspace{1cm}
HPO has a lot of structure $\leadsto$ treating it as generic black-box optimization task is unnecessarily general.

\end{frame}

\begin{frame}{Summary} 

Most important insights from this video:
\pause

\begin{enumerate}
    \item Bayesian optimization can be improved by exploiting the structure of the optimization problems.
    \pause
    \item HPO offers task similarity, prior information, and intermediate results.
\end{enumerate}
\pause

$\leadsto$ We derive HPO-specific assumptions that increase the efficiency of Bayesian optimization for HPO tasks.

\end{frame}
